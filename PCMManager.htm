1. Allgemein

Die Applikation PCM-Benutzerverwaltung dient zur Verwaltung von Benutzern und Benutzerrechten. Beim Programmstart wird das Dashboard angezeigt.

Das Dashboard zeigt 2 Auswertungen an:
Benutzer: Anzahl der Benutzer
Rechte: Anzahl der Rechte

2. Systemvoraussetzungen

  Systemvoraussetzungen
  Betriebssystem:  Windows 8, Windows 10, Windows 11
  Prozessor:  Prozessor mit 1 GHz oder höher
  Arbeitsspeicher:  2 Gigabyte
  Festplattenspeicher:  ca. 1 Gigabyte
  Grafikkarte:  DirectX 9 oder höher
  Display:  1280 x 800
  Internetverbindung:  Für die Aufrufe über die REST-API
  Freigabe Ports:  55700 TCP ausgehend
  SQL-Server:  MYSQL 5.x für 32-Bit
  MYSQL 8.x für 64-Bit

automatischer Login=automatischer Login

1.2. Benutzerverwaltung - Rechte

    - Lesen
    - Keine

2. Konfiguration

Im Modul Konfiguration kann das Design sowie auch der Fensterstil geändert werden. Designs werden sofort übernommen. Styles benötigen einen Neustart der Applikation.
Speichern=Speichern des Designs und Style
Design=Auswahl des Designs (Labels, Eingabefelder, Grids etc.)
Style=Auswahl des Stils (Formular)

3. Systeminfo

Im Modul Systeminfo werden die leistungsspezifischen Daten des Computers angezeigt. Sowie auch die Auslastung von Arbeitsspeicher und Prozessor.

4. Lizenz-Programminfo

Im Modul Lizenz-Programminfo werden die Versions- und Lizenzinformationen angezeigt.

5. Sprache wechseln

Im Modul Sprache wechseln kann die Sprache geändert werden.
Sprachen=Sprachenauswahl
Sprache wechseln=gewählte Sprache übernehmen

6. Abmelden

Unter Abmelden kann der User zur Laufzeit gewechselt werden.

7. Beenden

Im Modul Beenden wird die Applikation beendet.
